# Rendez-vous

## Requirements

[Expo Installation](https://docs.expo.dev/get-started/installation/)

## Installation

```shell
$ git clone https://github.com/kiki-le-singe/rendez-vous.git <name>
$ cd <name>
$ npm install or yarn
```

## Run

```bash
$ npm run ios
```

### Running tests

```bash
$ npm run test
```
